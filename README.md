## About Kinss
Kinss is a minimalist feed reader.

## Quickstart
1. `pip install -r requirements.txt`   
2. Run the script,default listen ip is `0.0.0.0` & port `5000`.

## Features
- [x] 多用户
- [x] 定时
- [ ] 已读/未读
- [ ] 加星
- [ ] 二维码
- [ ] 分类
- [ ] UI美化
